Downloads

Build from source if you can

Playdar is under active development, your are encouraged to compile it yourself from the source code if you are that way inclined.

Download Playdar Installers

These installers will give you a recent (17/Nov/2009) install of Playdar. They come with a stripped version of the Erlang runtime so you shouldn't need to install anything else to make them work. That said, these packages are rather new, so please be prepared to hop on IRC or the mailing list and ask for help if something doesn't work as expected :)
